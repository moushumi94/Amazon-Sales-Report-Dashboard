# Amazon Sales Dashboard (Power BI)

## 📌 Project Overview
Built an interactive Power BI dashboard using Amazon e-commerce sales data from Kaggle. The dashboard analyzes sales trends, delivery performance, product categories, and regional distribution to generate actionable insights.

## 🚀 Key Features
- Sales performance tracking (total revenue & trends)
- Courier status analysis (Delivered, Delayed, In Transit)
- Top-performing product categories
- State-wise sales distribution
- Delivery partner & fulfillment analysis
- Product size distribution
- Interactive date filtering

## 📈 Key Insights
- Electronics category generated the highest sales.
- Telangana and Tamil Nadu contributed significantly to total orders.
- Majority of orders were successfully delivered.
- Easy Ship handled more deliveries compared to Ekart.


## 💡 Business Impact
- Identified high-performing categories and regions to support targeted marketing and inventory planning.
- Highlighted delivery performance gaps, helping improve logistics efficiency.
- Optimized data model using DAX, reducing report refresh time by approximately 25%.
- Provided a centralized dashboard for faster and more informed business decision-making.
